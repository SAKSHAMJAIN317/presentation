Smart Attendance System

Image Verification & Metadata Based Authentication

Presented by: Saksham Jain

Introduction

Attendance monitoring is an essential part of academic and organizational environments.

Traditional systems rely heavily on manual entry or physical biometric devices.

These systems suffer from limitations such as proxy attendance, hardware dependency, and maintenance cost.

With digital transformation, automated and intelligent attendance solutions are required.

Introduction – Need for Smart Systems

Remote learning and hybrid work models require digital attendance verification.

Existing biometric solutions are not suitable for remote environments.

Image based verification provides a flexible and scalable alternative.

This project introduces a secure attendance system using image similarity and metadata validation.

Literature Review – Traditional Methods

  • Manual attendance registers are error prone and time consuming.
  • RFID based systems require physical infrastructure.
  • Biometric fingerprint systems face hygiene and hardware issues.
  • Smart card systems are vulnerable to proxy misuse.

Literature Review – Computer Vision Approaches

Recent research focuses on computer vision based attendance systems.

  • Face recognition using deep learning models
  • Real time camera based monitoring
  • Cloud based identity verification
  • AI powered attendance analytics

Literature Review – Research Challenges

  • Image spoofing using old photos
  • Duplicate attendance submission
  • Lighting and pose variation
  • Data privacy and storage concerns
  • False acceptance and rejection errors

Literature Review – Research Gap

Most existing systems focus only on face recognition accuracy.

They do not verify:

  • Image freshness
  • Metadata authenticity
  • Duplicate submissions

This project addresses these limitations using perceptual hashing and metadata validation.

Problem Statement

To design and implement a secure digital attendance system that prevents duplicate and fraudulent submissions by validating image similarity and metadata authenticity.

Objectives

  • Develop automated attendance marking system
  • Prevent duplicate image submission
  • Validate metadata timestamp of images
  • Maintain digital attendance records
  • Improve reliability of remote attendance monitoring

System Methodology – Overview

The system follows an image based verification pipeline.

User uploads an image → system processes → verifies → stores attendance.

Image Processing Module

  • Image uploaded by user
  • Perceptual hash generated
  • Hash represents visual identity
  • Compared with stored hashes

Similarity Detection

Perceptual hashing measures visual similarity between images.

  • Small hash difference = similar image
  • Threshold used to detect duplication
  • Prevents reuse of same image

Metadata Validation

  • Extract EXIF metadata
  • Check image capture date
  • Verify timestamp matches current date
  • Prevents old image reuse

Database Storage

  • User ID
  • Image hash
  • Attendance date

Records stored for verification and tracking.
